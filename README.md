# 🚀 NovaTech - Responsive Landing Page

A modern, responsive landing page built using HTML, CSS, and JavaScript as part of the SkillCraft Technology Web Development Internship.

## 📌 Project Overview

NovaTech is a fictional technology company website designed with a clean, modern user interface. The project demonstrates responsive web design principles and interactive UI elements.

## 🌐 Live Demo

🔗 https://novatech-landing-page-lasya.netlify.app/

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Fixed navigation bar
- Smooth scrolling navigation
- Hero section with call-to-action button
- Services section with interactive cards
- Features section
- Contact section
- Hover effects and subtle animations
- Modern dark theme UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
